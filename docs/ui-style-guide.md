# UI Style Guide

This document lists the colors currently used in the frontend styles.

## App.css

These are the main colors used by the journal page:

- Background: `#fbf8f0`
- Primary text: `#073c3b`
- Sidebar surface: `#fffdf7`
- Border color: `#eadfce`
- Main card/input surface: `#fffdf9`
- Input text: `#555`
- Neutral icon gray: `#66615d`
- Transport inactive: `#5c5955`
- Active transport background: `#fff1df`
- Active transport text and outline: `#9a3412`
- Heart accent: `#b45309`
- Photo icon: `#15758b`
- Photo caption: `#665d53`
- Music border: `#e6d9f4`
- Music surface: `#fff9ff`
- Purple accent: `#6f3bc4`
- Purple music text: `#30198b`
- Save button and status text: `#166534`

## index.css

These colors are also loaded globally and are still part of the app styles:

- Page background: `#fff8ef`
- Main body text: `#183d32`
- White surface: `#ffffff`
- Sidebar border: `#eee2d2`
- Active sidebar background: `#fff0dd`
- Sidebar active text: `#f47b38`
- Input border: `#eadfce`
- Upload border: `#e7cfae`
- Upload background: `#fffaf3`
- Button green: `#2f9b65`

## Notes

- Use the `App.css` colors for the current journal page UI.
- Keep new colors aligned with this palette unless there is a specific design reason to introduce a new one.
