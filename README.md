# Digital Travel Journal

## Overview

Digital Travel Journal is a web application that allows users to document and organize their travel experiences. Users can create journal entries, upload photos, track visited locations, and revisit memories through an interactive and user-friendly interface.

## Features

* User registration and login
* Create, edit, and delete journal entries
* Upload travel photos
* Organize trips and travel memories
* View visited locations on a map
* Search journal entries
* User profile management

## Workflow

Do not push directly to the main branch

### Starting a Task

1. Pull the latest changes:

```bash
git checkout main
git pull
```

2. Create a feature branch:

```bash
git checkout -b feature/taskname
```

### Saving Your Work

```bash
git add .
git commit -m "changes that you did"
git push -u origin feature/taskname
```

### Finishing a Task

Create a Pull Request from your feature branch into the main branch for review before merging.
